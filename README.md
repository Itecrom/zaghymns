# Zomba Assemblies of God App

**Connecting every member, everyday**

A Flutter mobile application designed to serve the Zomba Assemblies of God congregation with digital access to hymns, daily Bible reading plans, church announcements, and membership registration.

---

## 📱 About the App

**Zomba Assemblies of God App** is a cross-platform mobile app built with Flutter. It provides the ZAG congregation with:

- 📖 **Digital Hymn Book** — Access to Chichewa, Chitumbuka, and English hymns
- 📅 **Daily Bible Reading** — 2026 ZAG Annual Bible Reading Plan with Chichewa and English translations
- ⭐ **Favourites** — Save and quickly access your favourite hymns
- 🌓 **Dark / Light Mode** — Easy-on-the-eyes reading in any lighting condition
- 🔤 **Adjustable Font Size** — Global font size control for better readability
- 🌐 **Bible Reading Language** — Switch between Chichewa and English for daily verses
- 📝 **Membership Registration** — Register as a ZAG member directly within the app via an embedded WebView
- 💬 **Contact Lead Developer** — Quick WhatsApp access to the app's lead developer
- 📢 **Announcements** — Stay updated with the latest church news and events
- 🎵 **Nyimbo za Mbukhu** — All hymn collections in one convenient, expandable card

## ✨ Features

### Hymn Collections
- **Nyimbo za Mbukhu** — All hymn collections in one expandable card
  - **Chichewa Hymns** — Full collection of Nyimbo za Chitsitsimutso
  - **Chitumbuka Hymns** — Complete Sumu za Chitumbuka collection
  - **English Hymns** — Coming soon
  - **Favourites** — Personalised list of saved hymns

### Daily Bible Reading
- Follow the **2026 ZAG Annual Bible Reading Plan**
- Daily verse delivered automatically
- **Chichewa** translation via CLB (Chichewa Living Bible)
- **English** translation via WEB (World English Bible)
- Offline-capable calendar references stored in `adds/verses.json`

### Announcements
- **Firebase-powered announcements** — Real-time church news and updates
- Announcements disappear when new ones are posted
- Accessible from the home screen

### User Experience
- **Dark / Light theme** toggle in the Settings sidebar
- **Global font size** control (85%, 100%, 115%, 130%)
- **Bible reading language** selector (Chichewa / English)
- **Share the app** directly with friends and family
- **Splash screen** with animated logo and rotating credits
- **Update banner** — Notified when a new version is available

### Membership Registration
- In-app **WebView** registration form at `https://zombaassemblies.ct.ws/`
- No browser redirect — form loads seamlessly inside the app
- Campaign banner on the home screen for quick access
- Sidebar shortcut in Settings

### Contact & Support
- **WhatsApp integration** — Tap the Lead Developer card in the About dialog to send a pre-filled message

## 🚀 Getting Started

### Prerequisites

- **Flutter SDK** — [Install Flutter](https://docs.flutter.dev/get-started/install) (version compatible with SDK ^3.12.2)
- **Android Studio** or **VS Code** with Flutter extensions
- **Xcode** (for iOS builds on macOS)

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd zaghymn
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Generate launcher icons** (optional — icons are already generated)
   ```bash
   flutter pub run flutter_launcher_icons:main
   ```

4. **Run the app**
   ```bash
   flutter run
   ```

### Building for Release

#### Android
```bash
flutter build apk --release
# or for App Bundle
flutter build appbundle --release
```

#### iOS
```bash
flutter build ios --release
```

## ⚙️ Configuration

### Bible Reading API Keys

The app supports two Bible API providers:

1. **API.Bible** (preferred for Chichewa) — Requires an API key from [scripture.api.bible](https://scripture.api.bible/)
   - Set the key in `lib/main.dart` by replacing `_apiBibleKey`
   - Uses the `_chichewaBibleId` for Chichewa (Baibulo Lopatulika 2016)

2. **bible-api.com** (free fallback) — No API key required
   - Used automatically when API.Bible key is not configured
   - Supports CLB (Chichewa Living Bible) and WEB (World English Bible)

### Reading Language Preference

The user's Bible reading language preference is stored locally using `SharedPreferences`:
- Key: `reading_language`
- Values: `'chichewa'` or `'english'`
- Default: `'chichewa'`

### Theme Preference

Dark / Light mode preference is stored locally:
- Key: `night_mode_enabled`
- Default: `false` (light mode)

### Font Size

Global font size is stored locally:
- Key: `global_font_size`
- Values: `0.85`, `1.0`, `1.15`, `1.30`
- Default: `1.0`

## 📱 App Architecture

### Screens

| Screen | Purpose |
|--------|---------|
| `SplashScreen` | Animated logo + rotating credits on app launch |
| `HomeScreen` | Main dashboard with daily reading, Nyimbo za Mbukhu, announcements, and campaign banner |
| `HymnListScreen` | Searchable list of hymns for a selected collection |
| `HymnDetailScreen` | Full hymn view with lyrics, chords, and metadata |
| `AnnouncementsScreen` | Firebase-powered church announcements and news |
| `RegistrationWebViewScreen` | Embedded WebView for membership registration |
| `AppInfoDialog` | About dialog with app details and developer contact |

### Key Widgets

- **`_MiniCollectionCard`** — Compact half-width card for hymn collections
- **`_MembershipCampaignBanner`** — Promotional banner for membership registration
- **`_DrawerTile`** — Reusable sidebar menu item
- **`LogoMark`** — App logo widget used across splash screen, header, and drawer
- **`PremiumBackground`** — Themed background wrapper for screens

### Data Flow

```
User opens app
    ↓
SplashScreen (animated logo + credits)
    ↓
HomeScreen
    ├── Daily Reading Card (fetchTodaysVerse)
    │       ├── API.Bible (Chichewa, if key configured)
    │       ├── bible-api.com CLB (Chichewa fallback)
    │       └── bible-api.com WEB (English fallback)
    ├── Nyimbo za Mbukhu Card (expandable)
    │       ├── Chichewa Hymns → HymnListScreen
    │       ├── Chitumbuka Hymns → HymnListScreen
    │       ├── English Hymns → Coming Soon
    │       └── Favourites → HymnListScreen
    ├── Announcements Card → AnnouncementsScreen (Firebase Firestore)
    ├── Membership Campaign Banner → RegistrationWebViewScreen
    └── AppSettingsDrawer (sidebar)
            ├── Theme toggle
            ├── Language selector
            ├── Font size control
            ├── Share app
            ├── Membership Registration
            └── About dialog
```

## 🗂️ Database

The app uses a local SQLite database (`adds/chitsitsimutso.db`) to store hymn data. The database is bundled with the app and accessed via the `sqflite` package.

- **Hymn collections**: Chichewa, Chitumbuka, English
- **Each hymn contains**: number, title, lyrics (HTML), chords, searchable text

## 🌍 Localization

The app is primarily designed for a Malawian audience:

- **Primary language**: Chichewa (with Chitumbuka and English hymn collections)
- **Bible reading**: Chichewa (CLB) or English (WEB)
- **UI labels**: English with Chichewa context
